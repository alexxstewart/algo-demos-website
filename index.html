How It Works

The Demo above shows a common path finding algorithm in use. There are many different types of path finding algorithms which have specific use cases

In the demo above you were prompted to give and start and end node

We then iterate through the nodes in the tree and compute the shortest path between the start node and the end node.

Once we have visisted all the nodes we then terminate the algorithm
